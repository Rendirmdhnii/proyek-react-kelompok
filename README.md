## 🛠️ Stack Teknologi
- **Framework:** [ReactJS (Vite)](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
- **Version Control:** Git & GitHub

---

## 📖 Fitur Utama
- **Responsive Design:** Tampilan optimal di HP maupun Laptop.
- **Modern UI:** Menggunakan standar Tailwind v4 terbaru.
- **Routing:** Navigasi antar profil tanpa reload halaman (SPA).
- **Interactive:** Animasi transisi yang halus di setiap halaman profil.

## 🚀 Panduan Instalasi Lokal

Jika ingin menjalankan project ini di komputer lokal, ikuti langkah berikut:

1. **Clone Repository:**
   ```bash
   git clone [https://github.com/Rendirmdhnii/proyek-react-kelompok.git](https://github.com/Rendirmdhnii/proyek-react-kelompok.git) git clone

2. Masuk ke Direktori: cd proyek-react-kelompok

3. Install Dependencies: npm install

4. Jalankan Project: npm run dev

5. Buka http://localhost:5173 di browser Anda.

📋 Project ini mengikuti skema kolaborasi tim sebagai berikut:

main : Branch utama untuk kodingan final yang sudah stabil.

dev/sofyan : Branch pengembangan untuk profil Sofyan.

dev/desbellion : Branch pengembangan untuk profil Desbellion.

dev/krisna : Branch pengembangan untuk profil Krisna.
